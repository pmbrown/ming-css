# Ming-CSS

A lightweight color-scheme friendly style sheet that provides the basic building blocks for modern website content design including:

- Theme color-scheme variables.
- Reset/normalize styles for element-specific HTML. 
- Styles for commonly used components. 
- Content utilities and modifiers.
- Primary color variables and utility classes.

The styles are ready-to-use and customize by developers with any level of experience in writing CSS, the demo website (https://ming.pmbrown.net/) showcases the styles with HTML examples and basic usage info where required.

How to customize and/or expand the styles is up to the users imagination and experience, Ming-CSS is shared as-is without documentation and the website currently provided for demonstrative purposes only.
